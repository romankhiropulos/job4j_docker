# job4j_docker

## Описание проекта.
Изучение Docker 

#### 1) Скачиваем проект job4j_docker:
###### git clone путь_к_репозиторию

#### 2) Переходим в проект job4j_docker:
###### cd job4j_docker

#### 3) Запускаем команду сборки:
###### docker-compose build

#### 4) Запустим образ:
###### docker-compose run job4j_docker
